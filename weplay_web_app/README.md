# WePlay Web - تطبيق ويب يعمل على iPhone بدون تثبيت

هذه نسخة ويب لـWePlay يمكنك رفعها على أي استضافة ثابتة (GitHub Pages, Netlify, Vercel) أو فتحها محليًا على هاتفك.

### مميزات مدمجة في هذه النسخة:
- ملف شخصي بسيط (حفظ محليًا باستخدام localStorage)
- عملات (Coins) ومكافأة يومية
- قائمة ألعاب (تفتح داخل iframe) + مثال لعبة HTML5
- إرسال نتائج اللعبة للمتجر (postMessage)
- لوحة صدارة محلية
- تصميم متجاوب ويدعم RTL (العربية)

### كيف تستخدمها على iPhone (بدون Mac)
1. فك الضغط عن الملف ورفع المجلد إلى استضافة ثابتة (GitHub Pages أو Netlify أو Vercel) أو انسخ الملفات إلى مجلد في سيرفرك.
2. افتح الرابط في سفاري على iPhone أو أضف لصفحة الرئيسية (Add to Home Screen) ليشبه التطبيق.

### استضافة سريعة على GitHub Pages:
1. أنشئ repo جديد على GitHub.
2. ارفع محتوى هذا المجلد إلى الفرع `gh-pages` أو إلى الفرع الرئيسي مع تفعيل GitHub Pages من إعدادات الريبو.
3. افتح الرابط `https://<your-username>.github.io/<repo>/`

### تشغيل محليًا (لتجربة على الحاسوب):
- يمكنك فتح `index.html` مباشرة، أو تشغيل سيرفر بسيط:
  ```bash
  # Python 3
  python3 -m http.server 8000
  # ثم افتح http://localhost:8000 في المتصفح
  ```

### ملاحظات مستقبلية (لإضافة ميزات كاملة):
- تسجيل دخول حقيقي (Firebase Auth / OAuth) — يتطلب إعدادات خلفية.
- إشعارات Push — تتطلب HTTPS وService Worker + Server Key.
- حفظ نتائج مركزي (Server or Firebase) — لو أردت أحول التخزين المحلي إلى قاعدة بيانات، أقدر أعد لك النسخة.

إذا بدك أحطلك المشروع على GitHub Pages وأعطيك الرابط مباشرًا، قول لي "انشرلي على GitHub" وأنا أساعدك خطوة بخطوة.
